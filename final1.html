Meal Planning Guru Introduction

What is it?

  • MealPlanningIs.cool is an all-in-one recipe management program to handle home meal planning.
  • Provides ways for users to plan meals, look up recipes, upload their own, or shop for ingredients online through local marketplaces.
  • Can help plan meal prep, family dinner, or just find something new to try out in the kitchen.

Specific App Functionality

Calendar to allow users to schedule meals for each day.

Function like a “Home Page” of the app. Users open it and see what they have on the calendar as soon as it loads.

Allow users to link calendar to an external calendar.

Recipe Book to store both personally uploaded recipes and those found online.

Allow users to upload recipes, search for recipes online, import recipe from external sites.

Can keep track of what main ingredients users see when things maybe going bad/need to be used
